# 🌐 Landing Page – Teacher Cinthia Gabriel

Este projeto é uma **Landing Page profissional desenvolvida para divulgação de aulas de inglês**, com foco em conversão de alunos e experiência do usuário.

🔗 **Acesse o projeto online:**  
https://www.teachercinthiagabriel.com.br/

---

# 📌 Sobre o Projeto

Esta landing page foi desenvolvida para apresentar os serviços de uma professora de inglês e facilitar o contato com novos alunos.

O principal objetivo foi criar uma página:

- Simples
- Rápida
- Responsiva
- Estruturada para conversão

Este também foi **meu primeiro projeto de landing page publicado em produção**, marcando um passo importante na minha jornada no desenvolvimento web.

---

# 🎯 Funcionalidades

A página foi construída com foco em **UX/UI e conversão**, incluindo:

- ✅ Hero Section com proposta de valor clara
- ✅ Apresentação dos benefícios das aulas
- ✅ Estrutura pensada para guiar o usuário até o contato
- ✅ Botões estratégicos de chamada para ação (CTA)
- ✅ Layout totalmente responsivo
- ✅ Navegação simples e objetiva
- ✅ Carregamento rápido

---

# 💻 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias modernas de front-end:

- **HTML5** → Estrutura semântica da página  
- **CSS3** → Estilização e responsividade  
- **JavaScript** → Interatividade da interface  
- **Vercel** → Deploy e hospedagem

---

# 📱 Responsividade

A landing page foi projetada para funcionar perfeitamente em diferentes dispositivos:

- 💻 Desktop  
- 📱 Smartphones  
- 📲 Tablets
